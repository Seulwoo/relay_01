### II. 자연어 처리

> **A 기능 - 자연어 개발에 대해 기획하기.**

​    

사운드 클라우드의 앨범과 음악 소개를 자연어 처리를 통해 사용자가 노래 검색에 대한 접근과 공유를 쉽게 해준다.

자신의 감정을 한줄로 표현한 코멘트를 바탕으로 노래 검색 Ex) 오늘 비가 오니 우울해요.

​     
​     

### III. 비전 / 영상 처리

> **B 기능 - 비전 / 영상처리 기획하기.**

​     

공유한 웹캠을 바탕으로 사용자의 감정을 분석해서 분위기에 따른 노래를 추천해준다.

업로드한 일상 사진을 분위기에 맞는 BGM을 매칭시켜주거나 추천해줘서 커스텀 액자를 만들 수 있다.

앨범 커버를 분석해서 비슷한  카테고리로 묶어 사용자에게 추천.

​       
​     

### IV. 테이블 데이터 값

> **C 기능 - 표 형태의 데이터를 가지고 추천 기획하기.**

​     

사용자가 들었던 혹은 공유했던 음악을 바탕으로 노래 추천.
